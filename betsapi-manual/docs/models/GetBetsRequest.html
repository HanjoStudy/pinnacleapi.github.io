GetBetsRequest

  • betlist : String
    Type of bet list to return. Not needed when betids is submitted. => [ SETTLED, RUNNING, CANCELLED ]
    Enum:
    SETTLED
    RUNNING
    CANCELLED
  • fromDate : String
    Start date of the requested period. Required when betlist parameter is submitted. Difference between fromDate and toDdate canâ€™t be more than 30 days. Expected format is ISO8601 - can be set to just date or date and time.
  • toDate : String
    End date of the requested period. Required when betlist parameter is submitted. Expected format is ISO8601 - can be set to just date or date and time. toDate value is exclusive, meaning it cannot be equal to fromDate.
  • betids : List
    A comma separated list of bet ids. When betids is submitted, no other parameter is necessary. Maximum is 100 ids. Works for all non settled bets and all bets settled in the last 30 days.

    A comma separated list of Unique Ids to query eralier placed bets. If specified, is treated with highest priority, all other parameters are ignored. Maximum is 100 ids.
