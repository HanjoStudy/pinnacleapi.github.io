PlaceTeaserBetRequest

  • uniqueRequestId : String
    Client generated GUID for uniquely identifying the bet.
  • teaserId : Integer
    Unique identifier. Teaser details can be retrieved from a call to Get Teaser Groups endpoint.
  • oddsFormat : String
    Format the odds are returned in.
  • winRiskStake : String
    Whether the stake amount is risk or win amount.
  • stake : String
    Amount of the wager in the client's currency.
  • legs : List
    Collection of legs.
