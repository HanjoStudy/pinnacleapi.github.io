ParlayLegRequest

  • uniqueLegId : String
    Unique id of the leg. It's used to identify and match legs in the response.
  • lineId : String
    Line identification.
  • altLineId : String
    Alternate line identification.
  • pitcher1MustStart : String
    Baseball only. Refers to the pitcher for Team1. This applicable only for MONEYLINE bet type, for all other bet types this has to be TRUE.
  • pitcher2MustStart : String
    Baseball only. Refers to the pitcher for Team2. This applicable only for MONEYLINE bet type, for all other bet types this has to be TRUE.
  • sportId : String
    Sport identification.
  • eventId : String
    Event identification.
  • periodNumber : String
    This represents the period of the match. For example, for soccer we have: 0 - Game, 1 - 1st Half and 2 - 2nd Half
  • legBetType : String
    Only SPREAD, MONEYLINE and TOTAL_POINTS are supported.
  • team : String
    Chosen team type. This is needed only for SPREAD and MONEYLINE bet types.
  • side : String
    Chosen side. This is needed only for TOTAL_POINTS bet type.
