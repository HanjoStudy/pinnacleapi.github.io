Up 
get /v1/fixtures
Returns all non-settled events.Please note that it is possible that the event is in Get Fixtures response but not in Get Odds. This is happens when the odds have not been set yet for the event. Please note that it is possible to receive the same exact response when using "since" parameter. This is rare and can be caused by internal updates of event's properties. (fixturesGet)

Query parameters

sportId (required)
Query Parameter — The sportid for which to retrieve the fixutres. format: int32
leagueIds (optional)
Query Parameter — The leagueIds array may contain a list of comma separated league ids.
isLive (optional)
Query Parameter — To retrieve ONLY live events set the value to islive =1. Any other value will result in retrieval of events regardless of their Live status.
since (optional)
Query Parameter — This is used to receive incremental updates. Use the value of last from previous fixtures response. When since parameter is not provided, the fixtures are delayed up to 1 min to encourage the use of the parameter. format: int64
